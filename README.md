# Python Sales and Weather Analysis

## Описание проекта

Проект посвящен анализу продаж с использованием Python и исследованию влияния погодных условий на количество заказов.

Цель работы — выявить закономерности в продажах, определить аномальные значения и проверить возможную связь между температурой воздуха и покупательской активностью.

## Используемые инструменты

* Python
* Pandas
* NumPy
* Matplotlib

## Выполненные задачи

### Очистка и подготовка данных

* Выполнена загрузка и проверка данных.
* Преобразованы даты в корректный формат.
* Подготовлены данные для дальнейшего анализа.

### Анализ продаж

* Рассчитано ежедневное количество продаж.
* Построена временная динамика продаж.
* Выполнен поиск аномальных значений (выбросов).
* Определены наиболее популярные товары.

### Анализ погодных факторов

* Загружены данные о погоде.
* Рассчитана средняя температура по дням.
* Выполнено объединение данных о продажах и погоде.
* Проведен сравнительный анализ температуры и количества продаж.
